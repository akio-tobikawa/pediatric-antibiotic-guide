# 小児抗菌薬 用量ナビ

`index.html` をブラウザで開くと利用できます。検索欄に薬品名を入力し、体重（kg）を入れると、原本の mg/kg/日 から1日量の範囲を換算します。

## 収載範囲

現在は、提供された「15_表16_小児用量.pdf」の先頭ページ群（原本 p.1〜6）から、アミノグリコシド系、カルバペネム系、セファロスポリン系、モノバクタム系などの主要薬品を構造化しています。原本の疾患別条件、腎機能別条件、年齢区分、製剤濃度は、この画面の換算値だけで代替せず、必ず原本の該当ページと添付文書を確認してください。
